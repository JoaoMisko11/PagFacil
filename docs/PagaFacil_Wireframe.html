Entrar com Magic Link
Sem senha. Enviamos um link para seu email.
Primeira vez? Cria a conta automaticamente.
Decisão de Design
US-09 — Magic link via Resend. Zero senha = zero fricção.
Se o email não existe, cria conta automaticamente (merge login+signup).
Fluxo
1. Usuário digita email
2. Clica "Entrar com Magic Link"
3. Tela muda para "Verifique seu email"
4. Clica no link → sessão JWT 30 dias
Tech
NextAuth.js v5 + Resend
Free tier: 100 emails/dia (suficiente para 10 users)

1. Usuário toca no ✓ do card
2. Card some da lista (animação fade-out)
3. Toast aparece: "✅ [nome] marcado como pago"
4. Botão "DESFAZER" no toast (countdown 10s)
