PagaFacil

Contas a pagar simplificado para MEIs e pessoas fisicas no Brasil

Visao Geral

PagaFacil e um web app gratuito de gestao de contas a pagar. O usuario cadastra suas contas, visualiza um dashboard inteligente e recebe lembretes automaticos por email, Telegram e/ou push notification.

Funcionalidades Principais

Tudo que o usuario precisa para nunca mais perder um vencimento.

📋

CRUD de Contas

Cadastro completo com fornecedor, valor (R$), vencimento, categoria (Fixo, Variavel, Imposto, Fornecedor, Assinatura, Funcionario, Outro) e observacoes. Edicao, exclusao com soft delete e recuperacao.

📊

Dashboard Inteligente

Visao consolidada com cards de vencidas, vencem hoje, proximos 7 dias e pendente 30 dias. Totalizadores de valores pendentes e pagos com atualizacao em tempo real.

📅

Calendario Visual

Calendario mensal interativo com dots coloridos (vermelho = vencida, amarelo = hoje, azul = a vencer). Clique no dia para ver detalhes. Totalizadores por mes ao navegar.

🔔

Lembretes Automaticos

Notificacao D-1 (um dia antes do vencimento) via email, Telegram e/ou push notification no navegador — o usuario pode ativar multiplos canais simultaneamente. Cron job diario as 8h BRT via Vercel Cron. Push notifications usam Web Push API (VAPID) com auto-cleanup de subscriptions expiradas.

🔁

Contas Recorrentes

Pre-geracao de parcelas (90 dias a frente). Frequencia configuravel: semanal, quinzenal, mensal ou anual. Data de fim opcional. Tela de pagamentos com tabs por urgencia.

🤖

Bot Telegram

@pagafacil_bot com comandos: /contas (lista pendentes), /nova (cria conta inline), /pagar (marca como paga interativamente), /ajuda. Gestao completa sem abrir o navegador.

🔒

Autenticacao Multi-canal

Login via Google OAuth (1 clique), magic link por email (Nodemailer/Gmail), ou OTP via Telegram. Sessao JWT com 30 dias de duracao.

🌓

Dark Mode

Toggle light/dark/system sem flash (FOUC). Implementado com script inline no head e localStorage. Cores semanticas Tailwind para compatibilidade total.

📄

Importacao por Planilha

Upload de arquivo Excel (.xlsx, .xls) ou CSV para importacao em lote. Preview com validacao visual (linhas validas/invalidas), mapeamento flexivel de colunas e categorias. Ate 500 contas por importacao.

📝

Cadastro em Lote

Tabela editavel direto no app para cadastrar varias contas de uma vez. Adicione e remova linhas, preencha fornecedor, valor, vencimento, categoria e marque como recorrente com frequencia (semanal, quinzenal, mensal, anual) e data de fim opcional. Salve tudo com um clique. Ate 100 contas por vez.

💬

Feedback Widget

Botao flutuante para envio de feedback direto (feature, bug, outro). Dados salvos no banco para analise durante fase de validacao.

👪

Family Link

Compartilhe contas com outra pessoa via link de convite. Ambos veem e gerenciam as mesmas contas. Convite com expiracao, pagina dedicada /family para gerenciar membros e sair da familia.

⚡

UX World-Class (Fase 2)

Optimistic UI ao marcar como paga (feedback instantaneo), Suspense streaming no dashboard (3 boundaries com skeletons), loading indicators na busca, drag-and-drop real na importacao, tabela responsiva (cards no mobile), onboarding em 3 passos com lembretes, pagina de lixeira com restauracao, e hierarchy visual nos summary cards com icones e cores de urgencia.

📈

Diferenciacao (Fase 3)

Grafico de barras interativo (6 meses pago vs pendente, tooltips, eixo Y), smart insights baseados em padroes de contas recorrentes, quick actions no calendario (criar conta, marcar como paga inline), checklist de onboarding persistente (4 marcos), relatorio mensal por email/Telegram, categorias com icones e cores, atalhos de teclado (n=nova, /=buscar).

🎉

Delight (Fase 4)

Saudacao contextual por hora do dia e dia da semana, confetti CSS ao zerar contas pendentes, streak de pontualidade (badge motivacional), easter egg celebratorio ao pagar todas do mes, haptic feedback no PWA, transicoes suaves entre paginas, modo compacto vs confortavel com toggle na nav.

Fluxo do Usuario

Da primeira visita ate a gestao completa das contas.

1

Acesso & Login

Usuario acessa a landing page e escolhe: login com Google (1 clique), magic link por email, ou OTP via Telegram.

2

Onboarding

Na primeira vez, fluxo guiado em 2 passos: informar nome e cadastrar a primeira conta. Redirect automatico para usuarios novos.

3

Dashboard

Visao geral: resumo financeiro, graficos de tendencia, calendario com dots, insights e streak de pontualidade. Sem lista de contas — foco em entender a situacao.

4

Pagamentos

Contas organizadas por urgencia: vencidas, hoje, proximos 7 dias, futuras. Marcar como paga com 1 clique. Contas recorrentes geram proxima parcela automaticamente.

5

Contas (Gestao)

Cadastro e gestao de contas agrupadas por categoria. Contas recorrentes aparecem como entrada unica. Criar, editar, deletar, importar, cadastro em lote.

6

Lembretes & Telegram

Recebe lembrete D-1 por email, Telegram e/ou push notification (pode ativar todos). Pode gerenciar contas direto no Telegram com /contas, /nova e /pagar.

7

Family Link

Convida outra pessoa via link para compartilhar contas. Ambos veem e gerenciam as mesmas contas. Gerencia membros na pagina /family.

Seguranca & Boas Praticas

Decisoes de seguranca tomadas no projeto.

  • ✓ Autenticacao JWT com NextAuth v5 (sessao 30 dias)
  • ✓ Server Actions com validacao Zod em todas as mutations
  • ✓ Verificacao de userId em toda query (multi-tenant seguro)
  • ✓ Soft delete (dados recuperaveis por 30 dias)
  • ✓ CRON_SECRET + header x-vercel-cron para proteger API routes de cron
  • ✓ Webhook Telegram autenticado via secret_token
  • ✓ OTP rate limit (max 5 tentativas por codigo, max 3 ativos)
  • ✓ Tokens de convite familia criptograficamente seguros
  • ✓ Sanitizacao HTML em mensagens Telegram (anti-injection)
  • ✓ Importacao de planilha re-validada server-side
  • ✓ CSRF protection via allowedOrigins
  • ✓ callbackUrl encodado contra open redirect
  • ✓ Valores monetarios em centavos (sem float)
  • ✓ Datas em UTC no banco, Sao_Paulo na UI
  • ✓ Secrets em env vars (nunca no codigo)
  • ✓ onDelete Cascade em Bill e Feedback
  • ✓ Indices compostos para queries performaticas
  • ✓ Transacao atomica em markBillAsPaid + recorrente
  • ✓ Skeleton loading states no dashboard e lista de contas
  • ✓ Error boundary branded com retry (error.tsx)
  • ✓ Viewport permite zoom (WCAG 1.4.4)
  • ✓ Toast feedback em todas as acoes (pagar, deletar)
  • ✓ Erros inline (sem alert() nativo)
  • ✓ aria-labels em botoes de acao

Deploy & Infraestrutura

100% serverless, 100% gratuito.

⚡

Vercel (Hobby)

Deploy automatico serverless. Funcoes edge para middleware. Cron job diario para lembretes. URL: pagafacil.work

🐘

Neon PostgreSQL

Banco serverless na regiao sa-east-1 (Sao Paulo). Free tier com 512MB. Connection pooling via Prisma.

📧

Gmail SMTP

Envio de emails via Nodemailer + App Password. Limite ~500/dia no free tier. Usado para magic links e lembretes D-1.

🤖

Telegram Bot API

Bot @pagafacil_bot via webhook. Sem dependencias externas (fetch nativo). Gratis e sem limites de mensagens.
